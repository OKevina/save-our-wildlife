# Save Our Wildlife

A small promotional web page for a wildlife conservation organization, built with
**separated content** (HTML, CSS and JavaScript in their own files).

## Files
| File | Purpose |
|------|---------|
| [`index.html`](index.html) | Page structure and content |
| [`styles.css`](styles.css) | Styling, responsive design and accessibility |
| [`script.js`](script.js)   | Button behaviour (play / hide the video) |

## Features
- Heading **"Save Our Wildlife"**.
- Embedded video `wildlife.mp4` with native **playback controls**.
- A short **description** below the video.
- A **button** that plays the video, or hides it when it is already playing.
- **Accessibility**: skip link, semantic landmarks, ARIA labels, visible focus
  outlines, screen-reader-only headings, and `prefers-reduced-motion` support.
- **Responsive design**: fluid heading, flexible container, responsive video and
  a mobile breakpoint.

## Add your video
Place your video file named **`wildlife.mp4`** (and an optional `poster.jpg`)
in the project root next to `index.html`.
